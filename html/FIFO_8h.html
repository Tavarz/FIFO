Initializes a FIFO with size "size".

Adds an elemnt to the FIFO.

Initially checks if the chosen FIFO has been initialized, then checks if the FIFO is full with the help of the array that save all FIFO's sizes and tails. If it's full prints a message saying it's full. If it's not full adds the new element to the last position in the array and increments the tail of this FIFO.
